🧠 Full Recap: Linear SVM — Explained Mathematically & Intuitively

⚙️ 1. Problem Setup

  • A set of points xᵢ ∈ ℝᵈ
  • Each with a label yᵢ ∈ {−1, +1}
  • Goal: Find a hyperplane that separates the classes with the largest margin possible.

🧱 2. Hyperplane Equation

A hyperplane in ℝd is defined as: w · x + b = 0

  • w: normal vector (controls direction and tilt)
  • b: bias (offset)

The plane divides space into:

  • w · x + b > 0 ⇒ class +1
  • w · x + b < 0 ⇒ class -1

🧮 3. Classification Constraint

For perfect separation, we want:

yᵢ (w · xᵢ + b) ≥ 1

This ensures each point is on the correct side and outside the margin zone.

📏 4. Margin Definition

Two parallel margin planes:

  • w · x + b = +1
  • w · x + b = -1

Margin = 2 / ||w||

To maximize the margin, minimize ||w||.

🎯 5. Hard-Margin SVM (Perfect Separation)

Optimization problem:

minimize   (1/2) * ||w||²
subject to yᵢ (w · xᵢ + b) ≥ 1  ∀i

This works only if the data is perfectly separable.

❗ 6. Soft-Margin SVM (Real-world version)

When data isn’t perfectly separable, we introduce slack variables ξᵢ:

yᵢ (w · xᵢ + b) ≥ 1 − ξᵢ   with ξᵢ ≥ 0

We now minimize both margin size and total violations:

minimize   (1/2) * ||w||² + C ∑ ξᵢ
  • ξᵢ: how much point i breaks the margin rule
  • C: penalty for constraint violations
  • High C → strict, less tolerance
  • Low C → more flexible, allows errors

🔍 7. Prediction Rule

Once trained, we predict a new point x using:

ŷ = sign(w · x + b)

🧠 Summary of Key Concepts

Concept Formula Meaning
Hyperplane w · x + b = 0 Decision boundary
Margin 2 / ||w|| Space between margin planes
Constraint yᵢ (w · xᵢ + b) ≥ 1 Ensures correct classification
Slack ξᵢ ≥ 0 Allows violation (soft margin)
Regularization C min (1/2) * ||w||² + C ∑ ξᵢ Controls trade-off: margin vs violations
